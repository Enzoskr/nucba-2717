Tipos de Profesores

En nuestras escuelas, tendras, además de profesores en cada materia, tambien dispondras de un tutor para todos los problemas puntuales que tengan.

Prueba de profesores

Nuestros profesores, son puestos a prueba cada 2 años con un examen para tener en cuenta nuevos conocimientos y mejoras para la forma de enseñar.

Capacitación de profesores

Proveemos de capacitaciones para futuros estudios de los alumnos, materias especificas previas para ingresar a una facultad para tener conocimientos suficientes para ingresar de la mejor forma a ella.
